## 3. 本轮新增的重要执行规则

### 3.1 草稿 / 定稿节奏

用户经常会说：

- “先别作图，我们聊聊”：必须只聊天，不调用图片工具。
- “自己点评”：不要出新图，要对当前草稿做诚实评价。
- “重来”：重构草稿，不要在错误方向上小修。
- “别递归”：一次性按明确问题修，不要一轮一轮小幅递归。

### 3.2 横版电影海报规则

对于 EDG / iG / 后续暗黑破坏神2这类系列：

- 默认优先 16:9 横版桌面壁纸。
- 不要竖版卡牌合集。
- 不要十张都一个站姿。
- 每张都必须像独立电影镜头。
- 单人、双人、三人、全家福的关系层级要清楚。

### 3.3 角色辨识度优先于好看

用户多次强调：

- 第一优先级是角色辨识度。
- 脸、官方皮肤、武器、轮廓、英雄气质必须先对。
- 好看但不像原角色，不合格。
- 胸、身材、性感只能服务画面，不能压过角色身份。

### 3.4 文字与签名

对普通纯壁纸：尽量无字。  
对冠军皮肤系列：允许且需要文字，但必须克制、正确、有签名感。

冠军皮肤文字应包含：

- 选手名。
- iG / EDG 皮肤名。
- 选手签名风格笔迹。
- 冠军年份 / World Champions / MVP 等荣誉信息。

风险：模型文字容易乱码。正式发布前可考虑后期统一加字，但生成阶段必须预留并尝试形成签名视觉。
